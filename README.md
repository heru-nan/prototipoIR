# Prototipo Ing. Requisitos

## Descripcion
Buscador para las asignaturas y sus respectivos programas. 

## Comandos Importantes
Para ejecutar los comandos es necesario tener instalado [nodejs](https://nodejs.org/en/) en su dispositivo.

Dentro del repositorio, para instalar las dependencias: 
```
npm install
```

Para iniciar Prototipo: 
```
npm run dev
```

Finalmente dirigirse a `http://localhost:1234`
